
# Calculator App

A modern, responsive calculator built with HTML, CSS, and Vanilla JavaScript.

**Live Demo:** https://vv1705260-rgb.github.io/calculator-app/

## 🚀 Features

- **Basic Operations**: Addition, Subtraction, Multiplication, Division
- **Additional Functions**: Percentage, Clear, Delete
- **Keyboard Support**: Use your keyboard for all operations
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Error Handling**: Prevents divide-by-zero errors
- **Clean UI**: Modern gradient design with smooth animations

## 🛠️ Tech Stack

- **HTML5**: Semantic structure
- **CSS3**: Grid layout, Flexbox, Gradients, Animations
- **JavaScript ES6+**: Class-based calculator logic, Event handling

## 📋 How to Use

| Button | Action | Keyboard |
| --- | --- | --- |
| `0-9` | Input numbers | `0-9` |
| `.` | Decimal point | `.` |
| `+ - × ÷` | Basic operations | `+ - * /` |
